

# Ops Checklists

## New Page Checklist
- [ ] Page has a single clear purpose
- [ ] H1 matches the page title
- [ ] Disclosure added (if affiliate links exist)
- [ ] At least 2 internal links added
- [ ] One primary CTA
- [ ] Mobile-friendly spacing (short paragraphs)

## New Article Checklist
- [ ] Primary keyword chosen (see `06_seo-research.md`)
- [ ] Keyword appears naturally in:
  - [ ] Title
    - [ ] H1
      - [ ] First ~100 words
      - [ ] Clear H2 sections
      - [ ] Adds value before any affiliate mention
      - [ ] Links to:
        - [ ] One Education page
          - [ ] One Buying page (if relevant)
          - [ ] Conclusion includes a next step for the reader

## Publish Checklist (Before Push)
- [ ] No broken internal links
- [ ] Any pricing/claims have dates or context
- [ ] Spelling pass
- [ ] Disclosure present where needed
- [ ] Update Change Log (if structural changes)
