

# Style Guide

## Voice & Tone
- Plain English, beginner-friendly
- No hype, no fear-mongering
- Honest about risks and uncertainty
- “Teach first, sell second”

## Writing Rules
- Short paragraphs (1–3 lines)
- Use headings every ~200–300 words
- Use bullets for steps, checklists, comparisons
- Define terms the first time they appear

## CTA Rules (Calls to Action)
- One primary CTA per page
- CTAs should be helpful, not pushy
- Example CTA language:
  - “Compare reputable dealers”
    - “See current spot price”
      - “Learn coins vs bars”

## Affiliate/Disclosure Style
- Disclosure appears before first affiliate link
- Never hide affiliate links behind misleading buttons

## Formatting Conventions
- H1: page title
- H2: major sections
- H3: subtopics
- Use **bold** for key terms, not entire paragraphs

## Page Footer Must Include
- Affiliate disclosure link/section
- About / Contact
