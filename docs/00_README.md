
# Silver Site - Project README

## Goal
Build a simple, trustworthy silver-focused website that educates 
beginners, tracks silver pricing, and monetizes through affiliate 
products and related content.

## Target Audience
- Beginners interested in silver bullion.
- Casual investors wanting simple explanations.
- People checking silver prices and learning how to buy/sell.

## Monetization
- Affiliate links (bullion dealers, storage, accessories)
- Educational content supporting purchase decisions
- No hype, no scams, no pump language

## Project Status
- Phase: MVP build
- Focus: site structure, core pages, first content batch

## Repo Structure
- '/docs' -> planning, strategy, operations
- '/pages' -> static site pages (HTML or markdown)
- '/assets' -> images, charts, styles

## Current Focus
- Finalize sitemap
- Define monetization rules
- Draft first 10 content pieces
